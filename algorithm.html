ALGORITHM

Using computational methods to identify genes whose activity changes in colon cancer.

Section 1: Overview

The Dataset — Raw CEL Files

GSE44076 is a publicly available gene-expression dataset from the NCBI Gene Expression Omnibus. The data were collected from human colon tissues using the Affymetrix Human Genome U219 microarray platform.

Each CEL file represents one colon tissue sample and stores fluorescence-intensity measurements from thousands of DNA probes.

50 Healthy colon samples
98 Adjacent-normal samples
98 Colon tumor samples

Main comparison: 50 healthy samples versus 98 tumor samples.

Limma — Differential Gene Expression Analysis

Limma is a statistical method used to compare gene expression between two or more groups and highlight differences between them. In this project, it compares healthy tissue with colon cancer tissue using DNA microarray gene-expression data.

Log₂ Fold Change

Measures how much a gene is upregulated or downregulated in cancer.

log₂(Average Cancer ÷ Average Healthy)

A positive value means upregulated in cancer; a negative value means downregulated.

P-value

Estimates whether the observed difference between healthy and cancer samples is statistically significant.

Adjusted P-value

Accounts for testing thousands of genes simultaneously and helps control false discoveries. Genes with an adjusted p-value below 0.05 are commonly treated as significant.

The Limma Process

  1. Normalize the expression data.
  2. Fit a linear model for each gene.
  3. Compare tumor and healthy samples.
  4. Calculate log₂ fold change and statistical significance.
  5. Output significantly upregulated and downregulated genes.

Prepare the Data in R

  • Load the raw microarray dataset.
  • Remove control probes.
  • Normalize expression data using RMA (Robust Multi-array Average).
  • Correct background noise from the Affymetrix chips.
  • Adjust fluorescence intensities between samples to reduce technical bias.
  • Average probe measurements into expression values for each gene.

Analysis Outputs

The processed results are transformed into three visual summaries.

Volcano Plot

Shows genes with significant expression changes by combining log₂ fold change and statistical significance.

Heatmap

Uses Welch's t-test to select the top 50 genes and displays differences in gene activity between healthy and cancer samples.

PCoA

Shows how samples group based on similarities and differences in their gene-expression patterns.
